**E-Commerce Sales Analysis**



📌 About This Project

End-to-end data analysis on 110,000+ real 

Brazilian e-commerce orders.



🛠️ Tools Used

\- Python (Pandas, NumPy)

\- SQL (SQLite)

\- Power BI

\- Streamlit



📊 Key Findings

\- Revenue grew 8x from 2016 to 2018

\- São Paulo drives 40% of total revenue

\- Credit card used in 66% of all orders

\- Tuesday is highest revenue day

\- Average delivery time is 12 days
